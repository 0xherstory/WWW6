# Artistic Co-creation

## 项目简介
共识原野（Consensus Field）是一个基于区块链技术的生成式艺术共创平台。

## 目录结构
- src/contracts: 智能合约代码
- src/backend: 后端服务代码
- src/ui: 前端界面代码
- docs: 项目文档

By Candy, Chen

-	
基于艺术家生态的制度创建的“共创项目”
在传统语境下，艺术机构（美术馆、商业画廊）像是“过滤器”和“翻译官”筛选艺术家的同时，也可以决定谁可以展出，告诉观众该如何理解作品。这个项目运用web3去中心化和投票共创机制，对“艺术机构所为权利中介”的一次改变。
在当代艺术语境下，“观看”本身就是一种权力行为，传统机构通过高昂的票价、展墙、安保将观看这种行为仪式化、隔离化，作为观众我们被迫负担高昂的门票，来接受单方面的输出，观念赋予。
基于这种不平衡的现象，在这个项目里我们将会打破这种隔离，建立一种艺术家自发的公平的环境，和纯粹的共识性观看模式。
项目将由艺术家发起，由观众共建。将艺术直接与观众之间发生一个交互，每次产生的互动和增加都会被用来完善这一艺术项目，并被永久的记录在链上，每次观众互动所增加的部分将生成一个NFT，作为参与共创的证明。
