# 模拟币生 Coin Life

一个基于Web的加密货币交易模拟游戏，体验7天的币圈生活。

## 🎮 游戏简介

在这个像素风格的模拟游戏中，你将体验一个新币种从种子期到终局期的完整生命周期。通过7天的交易决策，感受币圈的疯狂与残酷。

### 🎯 游戏特色

- **7天生命周期**：从种子期到终局期，每天都有不同的事件和挑战
- **真实事件模拟**：基于真实币圈事件设计的新闻系统
- **像素艺术风格**：复古的8-bit视觉效果和音效
- **夜间动画**：实时价格变动可视化，配合弹幕系统
- **多样化结局**：根据你的表现获得不同的称号和评价

### 🎲 游戏玩法

1. **每日决策**：根据当天的新闻事件，决定买入、卖出或持有
2. **夜间观察**：观看价格实时变动，体验持仓的紧张感
3. **风险管理**：平衡收益与风险，避免爆仓
4. **策略优化**：学习择时、仓位管理等交易技巧

## 🚀 快速开始

### 在线游戏

直接访问 [GitHub Pages](https://yourusername.github.io/coin-life) 开始游戏

### 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/yourusername/coin-life.git
cd coin-life
```

2. 使用任意HTTP服务器运行：
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve .

# 或直接用浏览器打开 index.html
```

3. 在浏览器中访问 `http://localhost:8000`

## 🎯 游戏机制

### 交易系统
- **滑杆操作**：通过滑杆控制买入/卖出比例
- **资产计算**：实时计算现金和持仓价值
- **风险控制**：防止过度杠杆和无效操作

### 事件系统
- **分层事件**：不同天数对应不同类型的市场事件
- **随机性**：每次游戏都有不同的事件组合
- **影响度**：事件对价格的影响程度各不相同

### 评分系统
- **ROI计算**：基于最终资产的投资回报率
- **行为分析**：择时能力、风险控制、交易频率等
- **称号系统**：根据综合表现给出个性化称号
