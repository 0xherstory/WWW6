# 她的分叉宇宙 - 项目概述文档

## 一、项目背景（真实数据）

### 1.1 行业规模
- **市场规模**：2023年中国网络文学阅读市场规模达到404.3亿元
- **IP市场**：网络文学IP市场规模达到2605亿元，同比增长近百亿
- **用户规模**：截至2023年6月，中国网络文学用户达到5.37亿人
- **创作者规模**：超过700万网文作者

### 1.2 "远洋捕捞"事件——女性创作者的系统性困境

**什么是"远洋捕捞"？**

2024年6月起，安徽、兰州等地警方发起跨省执法行动，抓捕了超过50名女性耽美（BL）文学作者，被指控"制作和传播淫秽物品牟利"。这些作者中很多是女大学生或刚毕业的低收入女性，来自重庆、浙江、福建、云南等多个省份。

这一行动被称为"远洋捕捞"——地方政府跨省执法，像深海捕鱼一样"捕捞"女性创作者。法律界质疑其合法性，认为中国缺乏针对虚构文学作品的淫秽标准司法解释。

**"远洋捕捞"背后的深层问题：**

前安徽检察官沈良庆指出，这些跨省执法行动可能与地方政府财政困难有关，借助政治正确的理由获取经济利益——女性创作者辛苦创作，成果却被层层"捕捞"。

### 1.3 女性创作者面临的四大困境

#### ❌ 1. 人身安全威胁
- 跨省抓捕行动针对女性作者
- 实名制导致网络暴力和线下骚扰
- 创作内容被随意定义为"淫秽"，面临刑事风险

#### ❌ 2. 版权保护缺失
- 2022年有467件主要版权保护案例
- 通过信息网络传播的侵犯著作权犯罪呈上升趋势
- 超过80%的作者遭遇过不同程度的抄袭

#### ❌ 3. 平台分成不公
**付费阅读模式：**
- 传统平台：作者与平台五五分成，作者最多拿70%
- 晋江文学城：VIP章节0.03元/千字，五五分成后作者只拿0.015元/千字

**分销模式（更恶劣）：**
- 分销商拿走90%
- 分销平台拿5%
- 版权方（原创作者）仅拿5%

**底层作者困境：**
- 案例：35万字小说，一年收益仅3000元
- 大多数作者需要保持全职工作，靠热爱创作

#### ❌ 4. 价值被系统性低估
- 女频作品被平台算法降权
- 女性创作者缺乏话语权和议价能力
- 传统出版中，作者版税仅约10%，中间商层层盘剥

## 二、我们的解决方案：分叉叙事宇宙

### 2.1 核心定位

**一个用Web3技术保护女性创作者的去中心化共创小说平台**

我们不是在创造一个新的"平台"来压榨创作者，而是在构建一个**去中心化的生态系统**，让创作者真正拥有自己作品的所有权和收益权。

### 2.2 三大核心创新

#### 🔐 1. Web3版权保护

**技术实现：**
- 内容哈希上链：创建不可篡改的时间戳证明
- IPFS去中心化存储：内容永久保存，无法被删除
- AES-256加密：只有授权者能阅读
- NFT所有权凭证：每次转售都有5-10%永久版税

**对比传统：**
- 传统版权局登记：需要3个月 + 300元
- 我们的方案：3分钟 + 约0.5美元Gas费

#### 🎭 2. 匿名性保护

**解决"远洋捕捞"问题：**
- 钱包地址代替实名认证
- 创作者可以安全创作，不暴露真实身份
- 彻底杜绝人肉搜索和线下骚扰
- 防止跨省执法的滥用

**隐私保护机制：**
- 无需KYC（实名认证）
- 收益直接打入钱包，无需绑定银行卡
- 社区交互时完全匿名

#### 🌳 3. 共创分叉模式

**故事不是线性的，而是一棵"分叉树"：**
- 任何读者都能续写分支剧情
- 社区投票决定"正史"（Canon）
- 每个分支都有独立的NFT和版权
- 创作者共享故事宇宙的IP价值

**让故事拥有无限可能：**
- 去中心化的叙事结构
- 社区共同决定故事走向
- 每个创作者都能从IP中获益

## 三、商业模式与收益分配

### 3.1 创作者收益模式（4种收入来源）

#### 1. 💎 NFT首次销售（40%权重）
- 章节铸造为NFT
- 定价：0.001-0.01 ETH/章
- 前48小时独家阅读权
- **100%收益归创作者**（仅扣除Gas费）

#### 2. 🔄 永久版税（10%权重）
- 每次NFT转售，原创作者获得5-10%
- 这是Web3独有的机制，传统平台没有！
- 作品价值越高，长期收益越大

#### 3. 👁️ 阅读量奖励（30%权重）
- 来自平台DAO奖励池
- 按贡献度自动分配
- 透明、公平、不可篡改

#### 4. 💰 打赏（20%权重）
- 读者自愿打赏
- **100%归创作者**（平台仅收2.5%手续费）

### 3.2 平台收益模式

**我们不是"平台"，而是"协议"：**

1. **交易手续费**：2.5%（对比传统平台50-70%！）
2. **铸造费用**：每次NFT铸造收取约0.0005 ETH
3. **DAO治理Token**：$STORY代币增值

### 3.3 收益分配对比


**对比传统平台：**
- ❌ 传统平台：创作者拿30%（$6,000）
- ❌ 分销模式：创作者拿5%（$1,000）

## 四、技术架构

### 4.1 智能合约系统

#### 版权保护流程：
```
内容创作
  → AES-256加密
  → 上传IPFS（去中心化存储）
  → 内容哈希上链（不可篡改的版权证明）
  → 铸造NFT（访问凭证 + 所有权证明）
  → 自动分成（智能合约执行）
```

#### 访问控制：
```
读者购买NFT
  → 智能合约验证所有权
  → 获取解密密钥
  → 本地解密阅读

或

等待48小时后免费解锁
```

#### 收益分配：
```
智能合约自动执行
  → 透明、公平
  → 无需平台人工干预
  → 实时到账
```

### 4.2 技术栈
- **区块链**：Polygon（低Gas费，约$0.01/笔）
- **存储**：IPFS + Filecoin
- **智能合约**：Solidity
- **前端**：React + Web3.js
- **加密**：AES-256 + RSA
