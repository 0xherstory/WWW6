# 分叉叙事宇宙 - Web3共创小说平台

## 🌟 项目概述

基于你提供的参考图片（晋江作者页面 + 底特律：变人流程图），重新设计的Web3共创小说平台故事页面。实现了真正的"平行宇宙"视觉效果和专业的分支叙事体验。

## 🚀 预览方式

**服务器已启动，访问：http://localhost:8000**

## ✨ 核心功能重新设计

### 1. 故事卡片页面（参考晋江作者页面）
- **封面展示**：高质量故事封面图片
- **作者信息区**：
  - 作者头像、认证标识
  - 作者ID、积分、最新作品
  - 关注按钮和互动数据
- **故事简介**：作者个人介绍和作品描述
- **统计数据**：分享数、收藏数、META标识
- **核心操作**：阅读、点赞、续写三大按钮

### 2. 分支宇宙页面（参考底特律：变人）
- **宇宙背景**：
  - 动态星空效果
  - 星云渐变背景
  - 真正的科幻平行宇宙感觉
- **专业流程图**：
  - 类似底特律的节点设计
  - 带预览图的章节卡片
  - 智能连接线系统
  - 正史/分支的视觉区分

### 3. 节点设计系统
- **原创节点**：金色边框，特殊标识
- **正史节点**：红色边框，投票最高
- **分支节点**：蓝色边框，用户续写
- **续写节点**：虚线边框，带小鸟动画

### 4. 交互体验升级
- **页面切换**：故事卡片 ↔ 分支宇宙
- **动态连接线**：自动绘制节点间连接
- **悬停效果**：节点放大、发光效果
- **续写界面**：完整的AI辅助创作工具
